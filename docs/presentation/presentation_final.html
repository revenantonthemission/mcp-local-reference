mcp-local-reference

MCP server for Zotero references, PDF text, and figure cropping

Python 3.11+ FastMCP PyMuPDF ChromaDB Pillow MIT License

Designed for research-informed blog writing with Claude Desktop

Problem & Solution

The Problem

  • Academic references live in Zotero
  • Blog writing happens in Claude
  • Manual copy-paste of citations, text, figures
  • No semantic search across your library

The Solution

  • MCP server bridges Zotero ↔ Claude
  • 9 tools for search, read, crop, cite
  • Semantic search via vector embeddings
  • Figures returned as inline images

Getting Started

1. Install

git clone https://github.com/YOUR_USERNAME/mcp-local-reference.git
cd mcp-local-reference
pip install -e .

2. Configure Claude Desktop

{
  "mcpServers": {
    "local-reference": {
      "command": "/path/to/.venv/bin/python",
      "args": ["-m", "mcp_local_reference"]
    }
  }
}

3. Use

  • "Index my library for semantic search"
  • "Search for papers about transformer architectures"
  • "Crop figure 2 from this paper"
  • "Format a Harvard citation for this reference"
